Question-1: What are the different ways to select an element in the DOM?

getElementById(): Selects a single element by its unique ID.
getElementsByClassName(): Selects all elements with a specific class name.
getElementsByTagName(): Selects all elements with a specific HTML tag name.
querySelector(): Selects the first element that matches a specified CSS selector.
querySelectorAll(): Selects all elements that match a specified CSS selector.

Question-3: What is event delegation in the DOM??

Event delegation in the DOM is a technique where you attach a single event listener to a parent element, rather than adding individual event listeners to each of its child elements. This approach leverages the concept of event bubbling. Event Bubbling: When an event occurs on an element, it "bubbles up" through the DOM tree, triggering event listeners on its parent elements. How Delegation Works: Instead of assigning a listener to every child, you assign one listener to the parent. When an event happens on a child, it bubbles up to the parent. The parent's listener then checks the event.target property to determine which child element originated the event. Benefits: Performance: Reduces the number of event listeners, improving performance, especially with many child elements. Dynamic Elements: Handles events on dynamically added child elements without needing to add new listeners. Simplified Code: Makes event handling code cleaner and more maintainable.

Question-4: What is event bubbling in the DOM?

Event bubbling in the DOM is a process where an event first occurs in a child element, then reaches the parent element, and so on. Through this the parent elements can also handle that event.

Question-5: How do you create, add, and remove elements using JavaScript?

Document.createElement() is used to create elements with JavaScript, appendChild() is used to add, and removeChild() is used to remove.
